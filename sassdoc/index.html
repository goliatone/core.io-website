    button_color

    @mixin button_color($color_1: black, $color_2: white, $color_3: false) { ... }

    Description

    Mixin to color a button with an optional 5px 5px 5px box shadow.

    Markup has the following structure:

      <button class="btn-black">...</button>
  <a class="btn-black">...</a>
  <div class="btn-black">...<div>

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $color_1

    Button's text and border color.

    		Colorblack
    $color_2

    Button's background color.

    		Colorwhite
    $color_3

    Button's shadow color.

    		Colorfalse

    Example

    Creating .button

    .btn-black
  +button_color(#f8f8f8, #2c2c31, #212121)

    Output

    css for .btn-black

    .btn-black {
  color: #f8f8f8;
  background-color: #2c2c31;
  border-color: #f8f8f8;
  box-shadow: #212121;
}

    button_draw

    @mixin button_draw($w: 200px, $h: 50px, $border_w: 1px, $border_r: false) { ... }

    Description

    Mixin to draw a button with given width and height.

    Markup has the following structure:

      <button class="btn-big">...</button>
  <a class="btn-big">...</a>
  <div class="btn-big">...<div>

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $w

    Button's width.

    		Length200px
    $h

    Button's height.

    		Length50px
    $border_w

    Button's border width.

    		Length1px
    $border_r

    Button's border radius.

    		Lengthfalse

    Example

    Creating .button

    .button
  +button_draw(200px, 53px, 1px, 3px)

    Output

    css for .btn-big

      .btn-big {
    display: inline-block;
    width: 200px;
    height: 53px;
    line-height: 51px;
    border: 1px solid black;
    border-radius: 3px;
    text-align: center;
  }
  .button:hover {
    cursor: pointer; }
  }
